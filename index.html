⚙ Excel → JSON Converters

Convert Excel spreadsheets directly in the browser — no Python or Colab required.

🏗️

Hierarchy BOM Builder

Converts an Excel file with Level 1–10 columns into the Hierarchy JSON used by the HIERARCHY ASSEMBLY tab.

Required columns: Level 1 … Level 10, ID. Figura, Description, Fabricante/Fornecedor, Referência comercial, Código SAP, Qtd, Documentação, Image URL1, Image URL2, Image URL3
📦

Material Catalog Builder

Converts an Excel file with material catalog columns into the JSON used by the MATERIAL SEARCH tab.

Required columns: Centro, Material, Texto breve de material, Texto Longo, Depósito, Utilização livre
✅

Checklist Builder

Converts an Excel file with task items and CARRO columns into the JSON used by the CHECKLIST DIGITAL tab.

Required columns: Item, Local, Sistema, Subsistema, TIPO ATIVIDADE, DESCRIÇÃO CURTA, DESCRIÇÃO LONGA, REFERÊNCIA, HYPERLINK REFERÊNCIA, ACESSO, FERRAMENTA, MATERIAL, CARRO 1, CARRO 2 …

Como usar:
